# Print Image Editor

Browser-based image editor built for a printing-industry test task. Vue 3 + Vuetify 3 + Pinia + TypeScript.

## Run

```bash
npm i && npm run dev
```

## Features

- Upload an image
- Non-destructive crop (Cropper.js)
- Live brightness / contrast / saturation sliders
- View original / reset edits
- Export edited PNG
- **Bonus:** greyscale & sepia filters, export operations JSON for replay

See [NOTES.md](./NOTES.md) for design decisions.
